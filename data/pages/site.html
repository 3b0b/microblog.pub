About This Site

As indicated in the footer, my blog is using microblog.pub, a single-user, ActivityPub-powered microblog. I have made some customizations, some of which I intend to contribute upstream eventually if upstream wants them. At the time of this update, I have added a branch on my fork that I can try to remember to keep set up so that it can be used to find all my customized functionality. For example, at the moment, that branch includes the creation of my first custom pages and the handler for them, but not subsequent content updates (such as updating this page).

My Indieweb wiki user page will likely continue to have some information about how I use the software for as long as I continue to do so, although it may sometimes lag behind reality.
